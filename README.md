# WatchdogReport
## About:
WatchdogReport is a minecraft plugin that support's All Minecraft Versions For Paper/Spigot/Bukkit Servers Made By Herex_777.
This plugin is a Remake of Hypixel Report System.

## Features:
- Report GUI ✔
- Punishements commands ✔
- Like Hypixel ✔
